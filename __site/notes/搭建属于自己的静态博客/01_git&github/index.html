管理你的网页代码

你可以用Git工具管理你的网页代码, 并将它存储到Github在线仓库.

发布你的网页

有很多平台提供静态网页托管服务. 本教程中我们使用Netlify工具. 将它关联到你的Github仓库后, Netlify能实现网页的自动更新.
